# React Todo Desk (without CRA)

### Watch this
https://todo-desk.vercel.app/

## Stack:
* Typescript
* React
* MobX
* TailwindCSS v3
* Framer Motion
* React Hook Form
* Webpack & ESLint

## In the project directory, you can run:
### Development:

```Shell
yarn run dev
```

### Start:

```Shell
yarn run start
```

### ESLint format code:

```Shell
yarn run lint:fix
```
